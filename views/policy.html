Privacy Policy

Last Updated:

Introduction

This Privacy Policy explains how the Privacy Policy Summarizer ("we", "our", or "the Service") handles information when you use our web-based tool. We are committed to minimizing data collection while providing an effective privacy policy analysis service.

Summary

Privacy Policy Summarizer collects temporary session data for authentication and form input data for analysis, which is not stored after processing. The AI API processes submitted privacy policy content in real-time based on the selected processing speed.

Information Collection and Usage

Temporary Session Data

We use Deno KV OAuth to manage authentication sessions. This only stores:

  • A temporary session ID for Google authentication
  • No personal Google account information is permanently stored

Form Input Data

When you submit privacy policies for analysis, we temporarily receive your inputs for:

  • Age
  • Education level
  • Privacy understanding rating (1-10)
  • Privacy policy text content
  • Selected processing speed preference

This information is:

  • Used only for the immediate analysis request
  • Transmitted to OpenRouter AI API for processing
  • Not stored or retained after processing
  • Used to contextualize the AI analysis based on the input demographics

Technical Processing

  • Your submitted privacy policy content is processed in real-time through OpenRouter's AI API
  • The processing speed you select determines which AI model is used

Third-Party Services

Google OAuth

  • Used solely for authentication
  • Managed through Deno KV OAuth
  • Only session IDs are temporarily stored
  • No Google profile information is retained by our service

OpenRouter AI API

  • Processes privacy policy text for analysis
  • Receives user demographics to contextualize responses
  • No logs or copies of analyzed privacy policies are retained by us
  • Use of OpenRouter's AI API is governed by OpenRouter's Privacy Policy.

Data Storage

We intentionally minimize data storage:

  • No database of user information
  • No storage of submitted privacy policies
  • No retention of analysis results
  • Session IDs are temporary and cleared upon sign-out or expiration

Security

We protect your data during transmission by:

  • Using HTTPS for all communications
  • Securing API calls with appropriate authentication
  • Processing data in real-time without persistent storage

Your Rights

  • You can sign out at any time, clearing your session
  • You may use the service without concern for data retention, as we don't store your information
  • Please contact us with any privacy concerns

Children's Privacy

While our form accepts age inputs below 18 for research purposes, users must be 18 or older to use this service. By using our service, you confirm that you are 18 years of age or older.

Contact

For questions about the contents of this Privacy Policy, our data handling, or the Web interface in general, please contact gsrosato@wpi.edu.

Open Source Transparency

This service is on GitHub under the MIT license and is open source. You can verify our privacy practices by reviewing our code repository, which shows:

  • Temporary session handling
  • Real-time processing without retention
  • Clear data flow
